## Notas de estilo
- Números, fórmulas, unidades (kg CO₂, R²=0.89, <6%, litros, TWh) → **NUNCA se traducen ni reformatean**.
- Nombres propios de autores/estudios citados (Strubell et al., Patterson, Luccioni, IEA, Nature Comp. Sci.) → **se mantienen exactamente igual**.
- Rutas de archivo (`../assets/...`), clases CSS, IDs → **se mantienen intactas**.
- El inglés (junto con el español) es una **versión oficial validada**, no una traducción derivada — no requiere nota de traducción/disclaimer en las páginas.
